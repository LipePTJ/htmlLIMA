Lista de um checklist atividade

  1. Revisar se o código funciona sem erros no console
  2. Verificar se o nome do arquivo segue o padrão exigido
  3. Validar a indentação e organização do código (Clean Code)
  4. remover qualquer comentario no codigo
  5. enviar o codigo a alguem para verificar se acha erro
  6. verificar links quebrados ou caminhos de imagens incorretos
  7. Garantir que o arquivo foi salvo antes de compactar ou enviar
  8. Conferir se todos os requisitos da atividade foram atendidos
  9. fazer o commit e push para o repositório (se necessário)
  10. Anexar todos os arquivos necessários (HTML, CSS, Imagens)
